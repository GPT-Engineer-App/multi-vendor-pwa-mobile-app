## Setup

```sh
git clone https://github.com/GPT-Engineer-App/multi-vendor-pwa-mobile-app.git
cd multi-vendor-pwa-mobile-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
